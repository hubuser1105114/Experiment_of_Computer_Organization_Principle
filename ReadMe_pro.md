这是2021级广东工业大学计算机组成原理课程的实验，作者依然记得在做这个实验时候十分折磨。

语言使用了VHDL，而那时tiger也完全不教VHDL，想要看懂代码只能花不少时间去找和读VHDL的语法书。

更何况初学计算机组成原理，对计组是一知半解，Quartus软件也存在一定的使用障碍。

此repo尽可能地把作者当时完成的东西摆出来，可惜的是实验报告已经被tiger隐藏，众多细节作者也回想不起来了。

qpf文件即为entity的文件，使用Quartus打开。

pdf文件为VHDL电子书，读者可以在其中参考VHDL语法。

Quartus_pra1文件夹是f1实验，Quartus_pra3文件夹是f2实验, Quartus_pra4_test3文件夹是f3实验

Quartus_f4a_test2文件夹是f4a实验，Quartus_f4b_test1文件夹是f4b实验
